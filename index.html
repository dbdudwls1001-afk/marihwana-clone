💾 전체 데이터 백업 / 복원

이 기기에 저장된 모든 데이터(채널·소재·BGM·틱톡/인스타·API키·설정)를 하나의 JSON 파일로 백업하고, 다른 기기에서 복원할 수 있습니다.

📥 백업 (내보내기)
현재 데이터 전체를 .json 파일 하나로 다운로드합니다.
📤 복원 (가져오기)
백업 .json 파일을 선택하면 검증 후 현재 데이터를 전부 교체합니다. 복원 전 자동으로 현재 상태를 안전 백업(다운로드)합니다.
🔒 백업 파일에는 YouTube API 키가 포함될 수 있습니다. 타인과 공유 시 주의하세요.
📌 파일 형식 버전(schemaVersion)이 포함되어 향후 호환성을 판별합니다.

⚙️ YouTube API 키 설정

📖 사용설명서 (클릭하면 펼쳐집니다) ▼
🔑 STEP 0. 먼저 API 키를 설정하세요
① 오른쪽 위에 있는 🔑 API 키 설정 버튼을 클릭합니다.
② 팝업이 뜨면 YouTube API 키를 입력합니다.
　 → API 키가 없다면? 팝업 아래쪽 🔑 API 키 발급 방법을 펼치면 순서대로 안내가 나옵니다.
③ ✅ 저장 버튼을 누르면 끝!
　 → 저장 후 버튼 옆 ● 초록불이 켜지면 정상입니다.
💡 API 키는 내 브라우저에만 저장되며, 제작자나 다른 사용자에게 전송되지 않습니다. 단, 채널 정보를 불러올 때 Google(YouTube) 서버에는 사용됩니다. 공용 PC에서는 사용 후 🗑 삭제를 권장합니다.
📺 STEP 1. 채널 벤치마킹 하는 법
① 상단 탭에서 📺 채널 벤치마킹을 선택합니다.
② 채널 또는 쇼츠 링크 입력 칸에 유튜브 채널 주소를 붙여넣습니다.
　 → 예시: https://www.youtube.com/@핫하군
　 → 한글 주소도 OK! 주소가 길게 깨져보여도 그대로 붙여넣으면 됩니다.
③ 불러오기 버튼을 클릭하면 채널명, 구독자, 영상수, 조회수가 자동으로 채워집니다.
④ 아래 칸들을 채웁니다:
• 콘텐츠 유형 — 클릭하면 목록이 나옵니다. 직접 입력도 가능!
• 스크랩 이유 — 이 채널을 왜 스크랩했는지 메모
• 레퍼런스 — 참고할 영상이나 포인트
• 결 — 이 채널의 전체적인 느낌/방향성
⑤ 저장 버튼을 누르면 아래 표에 추가됩니다!
🎬 STEP 2. 소재 스크랩 하는 법
① 상단 탭에서 🎬 소재 스크랩을 선택합니다.
② 영상 링크 입력 칸에 유튜브 영상(쇼츠) 주소를 붙여넣습니다.
　 → 예시: https://www.youtube.com/shorts/F4F1H7Js3T4
③ 불러오기 버튼을 클릭하면 제목, 채널명, 업로드일, 조회수가 자동으로 채워집니다.
④ 아래 칸들을 채웁니다:
• 콘텐츠 유형 — 채널 벤치마킹과 동일한 유형 목록 사용
• 스크랩 이유 — 이 영상을 왜 스크랩했는지
• 내용 — 영상 핵심 내용 메모 (자유)
• 결 — 이 영상의 느낌/톤앤매너
⑤ 저장 버튼을 누르면 아래 표에 추가됩니다!
📂 STEP 3. 콘텐츠 유형 직접 만들기
① 콘텐츠 유형 선택 버튼을 클릭합니다.
② 맨 위에 있는 직접 입력 칸에 원하는 유형을 적고 추가를 누릅니다.
③ 추가한 유형은 목록에 바로 나타나고, 옆에 버튼들이 있습니다:
• ☆ / ★ — 즐겨찾기 (⭐ 켜면 맨 위에 고정!)
• ✏️ — 이름 수정
• 🗑 — 삭제
💡 YouTube 기본 카테고리 17개도 미리 들어있으니 골라 쓰셔도 돼요!
🔍 STEP 4. 검색 · 필터 · 정렬 · CSV 다운로드
🔍 검색 — 표 위 검색창에 키워드를 입력하면 채널명, 스크랩이유, 유형 등에서 즉시 필터링됩니다.
📁 유형 필터 — 검색창 옆 드롭다운에서 특정 유형만 골라볼 수 있습니다.
↕️ 정렬 — 표 헤더(채널명, 구독자, 조회수 등)를 클릭하면 오름차순/내림차순으로 정렬됩니다.
📥 전체 내보내기 — 현재 탭에 모은 데이터를 통째로 CSV 파일로 다운로드합니다.
✅ 선택 내보내기 — 원하는 항목만 골라서 내보낼 수 있어요. 누르면 각 줄 앞에 체크박스가 생기고, 체크한 것만 CSV로 나갑니다. 위에 선택 N / 전체 M개 숫자가 떠서, "30개만 제출" 같은 과제에서 개수를 딱 맞추기 좋아요. (검색·정렬해도 체크한 항목은 그대로 유지됩니다.)
📤 가져오기 — 다른 기기에서 내보낸 CSV 파일을 선택하면 데이터가 합쳐집니다. 같은 채널/영상은 자동으로 건너뛰어서 중복 걱정 없어요!
✏️ STEP 5. 저장된 항목 수정 · 삭제
표에서 각 항목의 오른쪽 액션 열에 3개 버튼이 있습니다:
• ⭐ — 즐겨찾기 토글
• ✏️ — 수정 팝업 열기 (유형, 스크랩이유, 레퍼런스, 결 수정 가능)
• 🗑️ — 삭제 (확인 팝업이 뜹니다)
💡 채널명/제목을 클릭하면 유튜브 새 탭으로 열립니다!

Q. 데이터가 어디에 저장되나요?
A. 내 브라우저(localStorage)에만 저장됩니다. 서버로 전송되지 않아요. 단, 브라우저 데이터를 지우면 같이 사라지니 중요한 데이터는 CSV로 백업해두세요!
Q. 다른 기기에서도 쓸 수 있나요?
A. 데이터는 기기별/브라우저별로 따로 저장됩니다. 하지만 📥 전체 내보내기로 CSV를 다운받아 카톡이나 메일로 보낸 뒤, 다른 기기에서 📤 가져오기를 하면 데이터를 옮길 수 있어요!
Q. API 사용량이 10,000을 넘으면?
A. 유튜브 API 무료 할당량이 소진되면 "불러오기"가 안 될 수 있어요. 매일 태평양시 자정(KST 오후 5시경)에 자동 초기화되니 다음날 다시 쓰시거나 다른걸로 갈아 끼우시면 됩니다.
Q. 한글 채널 주소가 깨져 보이는데?
A. https://www.youtube.com/@%ED%95%AB%ED%95%98%EA%B5%B0 이런 식으로 길게 나와도 그대로 붙여넣으면 됩니다. 자동으로 변환됩니다!
Q. CSV 파일을 열었더니 글자가 깨져 보여요 (□□□)
A. CSV는 눈으로 보는 파일이 아니라 데이터를 옮기는 파일이라 굳이 열어볼 필요가 없어요. 받은 파일은 그냥 📤 가져오기로 넣으면 되고, 내용을 보고 싶으면 가져오기 한 다음 표에서 확인하세요.
⚠️ 메모장·워드패드로 CSV를 열어서 "저장"을 누르면 파일이 손상될 수 있어요. 특히 내가 📥 전체 내보내기로 만든 백업 파일은 손상되면 되살릴 수 없으니 열지 마세요. (꼭 열어봐야 한다면 엑셀이나 구글시트로!)
Q. 데이터가 꼬였어요. 초기화하고 싶어요
A. 각 탭 툴바의 🧹 초기화 버튼을 누르면 그 탭 데이터만 비울 수 있어요. (채널·소재·BGM 각각 따로!) 누르기 전에 꼭 📥 전체 내보내기로 백업부터 하세요. 삭제하면 되돌릴 수 없습니다.
